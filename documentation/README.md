# Documentation

These guides are ordered as a tutorial path. Each page has a short description, related runnable feature commands, and previous/base/next navigation.

| Step | Guide | Description |
|------|-------|-------------|
| 1 | [Getting Started](getting-started/README.md) | Install the library, configure database access, and wire the core services. |
| 2 | [Entity Mapping](entity-mapping/README.md) | Map PHP classes to tables with attributes, primary keys, properties, indexes, and projections. |
| 3 | [Migrations](migrations/README.md) | Generate and apply schema changes from entity metadata. |
| 4 | [Relationships](relationships/README.md) | Define associations between entities, including polymorphic relationships. |
| 5 | [Query Builder](query-builder/README.md) | Compose filters, joins, aggregates, subqueries, and reusable Criteria. |
| 6 | [Pagination and Filtering](pagination-filtering/README.md) | Page through data and apply global filters such as soft delete. |
| 7 | [Lifecycle Callbacks](lifecycle-callbacks/README.md) | Hook into persistence, update, removal, and hydration events. |
| 8 | [Custom Types](custom-types/README.md) | Convert custom PHP values to and from database values. |
| 9 | [Transactions and Locking](transactions-locking/README.md) | Use transactions and row-level locks for consistent writes. |
| 10 | [Performance](performance/README.md) | Work with identity map, result cache, second-level cache, query logging, partial hydration, and batch iteration. |
| 11 | [Known Limitations](known-limitations/README.md) | Current library gaps and demo workarounds, grouped by area. |
