# Advanced Querying

Demonstrates where, whereIn, whereExists, joins, aggregates, subqueries, and Criteria.

**Read details:** [Query Builder](../../documentation/query-builder/README.md)

## Prerequisites

- Docker services running
- Migrations applied

## Run

```bash
bin/console app:example:advanced-querying
```

Or via Docker:

```bash
docker compose exec php bin/console app:example:advanced-querying
```

## Expected Result

Runs various query patterns and outputs results.
